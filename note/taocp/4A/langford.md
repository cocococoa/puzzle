# Langford対

## 理論

Thm1: `n = 4m + 1`, `n = 4m + 2` ではLangford対は存在しない

偶数は偶数番目と奇数番目に位置する。
奇数はどちらも偶数番目もしくはどちらも奇数番目に位置する。
`n = 4m + 1` の場合、偶数は `2m` 個存在するため、奇数が使える偶数番目の位置は `2m + 1` 個存在する。
これは偶数でないため、Langford対は存在しない。

Thm2: `n = 4m - 1`, `n = 4m` ではLangford対は存在する
